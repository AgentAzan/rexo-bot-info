Rexo Bot Commands

Here’s a full list of commands available in Rexo Bot. Supports both prefix (..) and slash (/) versions.

📊 Leveling

..rank / /rank — Show your or another user’s level & XP.
..leaderboard / /leaderboard — Show the server’s top 10 users by XP.
..enableleveling / /enableleveling — Enable the leveling system.
..disableleveling / /disableleveling — Disable the leveling system.

⚔️ Moderation

..clear / /clear — Delete messages.
..kick / /kick — Kick a user.
..ban / /ban — Ban a user.
..mute / /mute — Timeout a user.
..unmute / /unmute — Remove timeout.
..warn / /warn — Warn a user.
..unwarn / /unwarn — Remove last warning.
..warnings / /warnings — Show a user’s warnings.

🔐 Channel & Role Management

..lock / /lock — Lock a channel.
..unlock / /unlock — Unlock a channel.
..slowmode / /slowmode — Set slowmode.
..unslowmode / /unslowmode — Disable slowmode.
..move / /move — Move a user to a voice channel.
..nick / /nick — Change a user’s nickname.
..roleinfo / /roleinfo — Show info about a role.
..channelinfo / /channelinfo — Show info about a channel.

⚙️ Server Settings

..prefix / /prefix — Change the bot’s prefix.
..setwelcome / /setwelcome — Set the welcome channel.
..log / /log — Set the moderation log channel.

🎭 Fun & Utility

..ping / /ping — Check bot latency.
..uptime / /uptime — Show bot uptime.
..avatar / /avatar — Show a user’s avatar.
..userinfo / /userinfo — Show information about a user.
..serverinfo / /serverinfo — Show information about the server.
..meme / /meme — Fetch a random meme.
..test / /test — Simulate welcome message.
