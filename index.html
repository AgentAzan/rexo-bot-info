📊 Leveling

Track activity and reward users with XP and levels.

  • ..rank / /rank → Show user’s level & XP.
  • ..leaderboard / /leaderboard → Top 10 users in server.
  • ..enableleveling / /enableleveling → Enable leveling (admin).
  • ..disableleveling / /disableleveling → Disable leveling (admin).
×

⚔️ Moderation

Powerful moderation to keep your server safe.

  • ..kick / /kick → Kick a user.
  • ..ban / /ban → Ban a user.
  • ..mute / /mute → Timeout a user.
  • ..unmute / /unmute → Remove timeout.
  • ..warn / /warn → Warn a user.
  • ..unwarn / /unwarn → Remove last warning.
  • ..warnings / /warnings → View warnings.
