📜 Rexo Commands

Browse all commands available in Rexo, grouped by category.

📊

Leveling

  • ..rank / /rank → Show user’s level & XP.
  • ..leaderboard / /leaderboard → Top 10 XP leaderboard.
  • ..enableleveling / /enableleveling → Enable leveling system (admin).
  • ..disableleveling / /disableleveling → Disable leveling system (admin).
⚔️

Moderation

  • ..kick / /kick → Kick a user.
  • ..ban / /ban → Ban a user.
  • ..mute / /mute → Timeout a user.
  • ..unmute / /unmute → Remove timeout.
  • ..warn / /warn → Warn a user.
  • ..unwarn / /unwarn → Remove last warning.
  • ..warnings / /warnings → View warnings for a user.
  • ..clear / /clear → Delete messages.
🔧

Utility

  • ..ping / /ping → Check latency.
  • ..uptime / /uptime → Show bot uptime.
  • ..avatar / /avatar → Show user avatar.
  • ..userinfo / /userinfo → Show info about a user.
  • ..serverinfo / /serverinfo → Show server stats.
  • ..meme / /meme → Fetch a random meme.
  • ..test / /test → Simulate welcome message.
🔒

Channel & Role

  • ..lock / /lock → Lock a channel.
  • ..unlock / /unlock → Unlock a channel.
  • ..slowmode / /slowmode → Set slowmode.
  • ..unslowmode / /unslowmode → Disable slowmode.
  • ..move / /move → Move a user to voice channel.
  • ..nick / /nick → Change a user’s nickname.
  • ..roleinfo / /roleinfo → Show role info.
  • ..channelinfo / /channelinfo → Show channel info.
⚙️

Server Settings

  • ..prefix / /prefix → Change prefix for server.
  • ..setwelcome / /setwelcome → Set welcome channel.
  • ..log / /log → Set log channel.
