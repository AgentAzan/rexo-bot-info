Rexo Bot - Privacy Policy

1. Information We Collect

Rexo does not collect or store personal information. The bot only processes limited data required to function (such as usernames, IDs, and messages necessary for moderation, logging, and leveling features). No sensitive data is permanently stored.

2. How We Use Data

The data processed by Rexo is used only for moderation actions, server configuration (like welcome channels, prefixes, and logs), and fun/utility features (such as memes, XP leveling, and leaderboards). Any temporary data is cleared regularly.

3. Data Sharing

We do not sell, share, or distribute user data to third parties. Data stays within Discord’s platform and is only visible to authorized server staff.

4. User Rights

If you want to remove Rexo’s access to your data, you can kick or ban the bot from your server at any time. This will stop all data processing immediately.

5. Contact

If you have questions or concerns about this policy, please contact the bot developers through the Support Server.
